==============================================================================
anki.nvim                                                                 *anki*

 anki.nvim — Create, review, and manage Anki flashcards directly from Neovim.

 Features:
   • Browse decks and notes in a 3-pane split layout
   • Create, edit, send, and delete notes without leaving Neovim
   • Pull existing notes from Anki for editing
   • Deck management (create, rename, delete, switch profile)
   • Integrates with AnkiConnect (requires Anki running locally)

 Quick start: >
   require("anki").setup()
 <
 This opens a normal-mode mapping (default `<leader>a`) and/or the
 `:Anki` user command to launch the UI.

M.setup({opts})                                                     *anki.setup*
     Sets up the plugin with user configuration, keymaps, and commands.
     When `default_mappings` is true (the default), a normal-mode mapping for
     `prefix` (default `<leader>a`) is created to open the Anki UI.
     When `create_user_commands` is true (the default), the `:Anki` user
     command is created.

    Parameters: ~
        {opts}  (AnkiConfig|nil)  Optional configuration table (see |anki-config|).

    Usage: ~
>lua
        require("anki").setup({ url = "http://localhost:8765" })
<

Configuration                                                      *anki.config*

 Configuration module for anki.nvim.

 This module holds the default options and merges user-provided overrides
 via `setup()`. See |AnkiConfig| for the full list of options.

AnkiConfig                                              *anki.config.AnkiConfig*

    Fields: ~
        {url}                   (string)        AnkiConnect server URL.
        {timeout}               (number)        HTTP request timeout in milliseconds.
        {prefix}                (string)        Keymap prefix to open the Anki UI (default: `"<leader>a"`).
        {default_mappings}      (boolean)       Whether to auto-set the prefix keymap.
        {gui_browse_enabled}    (boolean)       Whether to open Anki GUI when sending/pulling notes.
        {create_user_commands}  (boolean)       Whether to create the `:Anki` user command.
        {mappings}              (AnkiMappings)  Buffer-local keymappings for deck, note, and editor panes.


AnkiMappings                                          *anki.config.AnkiMappings*

    Fields: ~
        {deck}    (AnkiDeckMappings)    Deck pane keymaps.
        {note}    (AnkiNoteMappings)    Note pane keymaps.
        {editor}  (AnkiEditorMappings)  Editor pane keymaps.


AnkiDeckMappings                                  *anki.config.AnkiDeckMappings*

    Fields: ~
        {show_help}       (string)  Show Help (default: `"?"`)
        {close}           (string)  Close (default: `"q"`)
        {select_deck}     (string)  Select deck (default: `"<CR>"`)
        {delete_deck}     (string)  Delete deck (default: `"d"`)
        {create_deck}     (string)  Create deck (default: `"c"`)
        {add_note}        (string)  Add note (default: `"a"`)
        {rename_deck}     (string)  Rename deck (default: `"m"`)
        {gui_deck}        (string)  Open in Anki GUI (default: `"o"`)
        {refresh_decks}   (string)  Refresh decks (default: `"r"`)
        {switch_profile}  (string)  Switch profile (default: `"p"`)


AnkiNoteMappings                                  *anki.config.AnkiNoteMappings*

    Fields: ~
        {show_help}          (string)  Show Help (default: `"?"`)
        {close}              (string)  Close (default: `"q"`)
        {edit_note}          (string)  Edit note (default: `"<CR>"`)
        {delete_note}        (string)  Delete note (default: `"d"`)
        {gui_note}           (string)  Open in Anki GUI (default: `"o"`)
        {show_all_notes}     (string)  Show all notes (default: `"a"`)
        {refresh_notes}      (string)  Refresh notes (default: `"r"`)
        {move_note_to_deck}  (string)  Move note to another deck (default: `"m"`)


AnkiEditorMappings                              *anki.config.AnkiEditorMappings*

    Fields: ~
        {send_note}    (string)  Send note to Anki (default: `"<leader>w"`)
        {pull_note}    (string)  Pull note from Anki (default: `"<leader>p"`)
        {delete_note}  (string)  Delete note (default: `"<leader>r"`)
        {kill_note}    (string)  Close note buffer (default: `"<leader>k"`)
        {show_help}    (string)  Show Help (default: `"?"`)


M.defaults                                                *anki.config.defaults*

    Type: ~
        (AnkiConfig)


M.options                                                  *anki.config.options*

    Type: ~
        (AnkiConfig)


M.setup({opts})                                              *anki.config.setup*
     Merge user options with defaults (deep-extend).

    Parameters: ~
        {opts}  (AnkiConfig|nil)  Partial or full configuration table to override defaults.

    See: ~
        |AnkiConfig|
